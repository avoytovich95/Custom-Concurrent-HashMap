Benchmark Explanation

Benchmarks were performed with 5 warmup iterations, and 15 test iterations. The wold server was able to handle a higher thread count, so 32 threads were used in the benchmark. The desktop however only had 8 cores to work with so a lower thread count of 16 was necessary. The data being displayed is the throughput benchmark mode, and recorded the data in operations per millisecond.
