Project Description

The main purpose of this project was to create a custom concurrent data structure, and benchmark it against an existing equivalent. For this project, a concurrent hashmap was created, and benchmarked against the existing concurrent hashmap found in the java concurrency library. To run the benchmarks, the JMH microbenchmarking library was used, and stored the data in the CSV format.
